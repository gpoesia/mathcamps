Problem: Harvey has 70 roller skates in his shop. If 42 of those roller skates are for children, how many roller skates does Harvey have for adults?

Gemma 2B: 

To determine how many roller skates Harvey has for adults, we need to subtract the number of roller skates for children from the total number of roller skates.

Given:
- Total roller skates = 70
- Roller skates for children = 42

We calculate the number of roller skates for adults as follows:
\[ \text{Roller skates for adults} = \text{Total roller skates} - \text{Roller skates for children} \]
\[ \text{Roller skates for adults} = 70 - 42 \]
\[ \text{Roller skates for adults} = 28 \]

### Answer: 28
