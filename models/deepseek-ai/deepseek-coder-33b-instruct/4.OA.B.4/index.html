A factor pair of a number is a pair of numbers that multiply to give the original number. For example, the factor pairs of 12 are (1,12), (2,6), (3,4), and (4,3), (6,2), and (12,1).

Here is a Python code to compute all the factor pairs of the number 59:

```python
def factor_pairs(n):
    factors = []
    for i in range(1, n + 1):
        if n % i == 0:
            factors.append((i, n // i))
    return factors

print(factor_pairs(59))
```

def factor_pairs(n):
    factors = []
    for i in range(1, n+1):
        if n % i == 0:
            factors.append(i)
    pairs = []
    for i in factors:
        for j in factors:
            if i * j == n:
                pairs.append((i, j))
    return pairs
