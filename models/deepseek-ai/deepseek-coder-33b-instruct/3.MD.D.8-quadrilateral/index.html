Problem: The perimeter of a rectangular swimming pool is 324 meters. The length of the pool is 83 meters. What is the width of the pool?

Gemma 2B: 

The perimeter of a rectangle is given by the formula 2*(length + width).
So, 324 = 2*(length + width)
Solving for width gives: width = 324/2 - length
Substituting the given length of 83 meters, we get: width = 324/2 - 83
Therefore, width = 324/2 - 83 = <<324/2 - 83=108>>108 meters.
