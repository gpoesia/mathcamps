Problem: In a camping trip, the vertical height of a tent is half of its diagonal length. If the diagonal length of the tent is 14 feet, what is the vertical height of the tent?

Gemma 2B: 

  Reasoning: Let's denote the base of the tent as "b" and the vertical height as "h". Since the diagonal length is the square root of (b^2 + h^2), we can write the equation:

14^2 = b^2 + h^2

Solving for h:

h^2 = 14^2 - b^2
h = sqrt(14^2 - b^2)

Since the height is half of the diagonal, we know that h = 7. So, we can set up the equation:

7 = sqrt(14^2 - b^2)

Squaring both sides:

49 = 14^2 - b^2
b^2 = 14^2 - 49
b^2 = 196 - 49
b^2 = 147
b = sqrt(147)

### Answer: The vertical height of the tent is 7 feet.

Expected answer: 7

Correct? Yes
