Problem: A king has decided to build a number of castles in his kingdom. If each castle requires 5 builders and there are 15 builders in total, how many castles are being built?

Gemma 2B: 

  Reasoning: Each castle requires 5 builders, so the number of castles being built is equal to the total number of builders divided by the number of builders required for each castle.

Total number of builders = 15
Number of builders required for each castle = 5

Number of castles being built = Total number of builders / Number of builders required for each castle
= 15 / 5
= 3

Therefore, the king is building 3 castles.
